# auth-script
This project demonstrate the script execution on providing valid key.

# Steps to start project.

1. Run command `python manage.py runserver` to start development server.
2. Server should have started on port `8000` at `http://127.0.0.1:8000/`
3. Navigate to `keywithscript\templates\home.html` and add key `TEST_KEY` to script parameter.
4. This will trigger the alert.
